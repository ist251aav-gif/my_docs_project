# Приклади використання

Наша програма підтримує базові арифметичні операції. Нижче наведено приклади того, як використовувати функції додавання та віднімання.

### Додавання
Щоб додати два числа, використовуйте наступний код:

```python
# add.py
def add(a, b):
    return a + b

print(add(5, 2))
```

Для віднімання чисел використовуйте цей приклад:

```python
# subtract.py
def subtract(a, b):
    return a - b

print(subtract(10, 4))
```
